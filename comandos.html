Sistema de moderação

O sistema de moderação possui várias funções úteis para ajudar a gerenciar um servidor config. Ele inclui um comando para configurar o servidor e também comandos como ban, kick e limpar para ajudar a manter a ordem e a organização no servidor.

Sistema de economia

O sistema de economia possui vários comandos úteis para gerenciar dinheiro. Ele inclui comandos para transferir dinheiro entre usuários, pegar dinheiro diariamente e explorar para ganhar (ou perder) uma quantia de dinheiro. Além disso, há um comando especial de perfil que permite aos usuários visualizar suas informações financeiras.
